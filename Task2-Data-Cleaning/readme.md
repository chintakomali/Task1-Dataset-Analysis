# Task 2 - Data Cleaning & Missing Value Handling

## Objective
Clean the Titanic dataset by identifying and handling missing values.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed

1. Loaded Titanic dataset.
2. Identified missing values using `.isnull().sum()`.
3. Visualized missing values using a heatmap.
4. Filled Age column missing values using Median.
5. Filled Embarked column missing values using Mode.
6. Removed Cabin column due to excessive missing data.
7. Saved cleaned dataset.

## Results

- Missing values successfully handled.
- Dataset prepared for Machine Learning.
- Cleaned dataset exported as CSV.
