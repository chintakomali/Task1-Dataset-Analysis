# Task 3 - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis on the Titanic dataset to identify patterns, relationships, and insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Cleaned Titanic Dataset

## Analysis Performed

### Univariate Analysis
- Histogram of Age distribution
- Boxplot of Fare distribution

### Bivariate Analysis
- Survival count based on gender
- Scatter plot between Age and Fare

### Correlation Analysis
- Generated correlation matrix
- Visualized using heatmap

### Pairplot
- Visualized relationships among numerical variables

## Key Insights
- Females had a higher survival rate.
- Higher fare passengers had better survival chances.
- Fare column contains outliers.
- Age distribution is approximately normal.

## Conclusion
EDA helped identify trends, relationships, and patterns in the dataset before machine learning modeling.
