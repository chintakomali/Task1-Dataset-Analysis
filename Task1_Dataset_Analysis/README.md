# Task 1 - Understanding Dataset & Data Types

## Objective
Understand the structure of a dataset and identify different data types.

## Tools Used
- Python
- Pandas
- NumPy
- VS Code

## Dataset Used
Titanic Dataset

## Steps Performed
1. Loaded dataset using Pandas.
2. Examined dataset structure using .info().
3. Generated statistical summary using .describe().
4. Identified numerical columns.
5. Identified categorical columns.
6. Checked missing values.
7. Displayed unique values in categorical columns.

## Findings
- Numerical columns:
  - PassengerId
  - Survived
  - Pclass
  - Age
  - SibSp
  - Parch
  - Fare

- Categorical columns:
  - Name
  - Sex
  - Ticket
  - Cabin
  - Embarked

- Missing values found in:
  - Age
  - Cabin
  - Embarked

## Conclusion
Successfully explored the dataset and understood the various data types and structure.
